## Как пользоваться

* В каждой папке группы есть свой `README.md` со списком вопросов-ссылок.
* Файлы вопросов имеют формат `NN-краткое-название.md`.
* Внутри файла: тезисы → развёрнутый ответ → примеры кода (где уместно).
